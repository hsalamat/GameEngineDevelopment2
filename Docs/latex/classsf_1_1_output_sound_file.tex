\doxysubsection{Detailed Description}
Provide write access to sound files. 

This class encodes audio samples to a sound file.

It is used internally by higher-\/level classes such as \mbox{\hyperlink{classsf_1_1_sound_buffer}{sf\+::\+Sound\+Buffer}}, but can also be useful if you want to create audio files from custom data sources, like generated audio samples.
